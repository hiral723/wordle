# 🟩 Wordle Clone

A simple and interactive Wordle-style word guessing game. Try to guess the hidden word within limited attempts using logic and letter hints.

---

## 🚀 Features

- Classic Wordle Gameplay: Guess the word in limited tries  
- Color Feedback System:  
  - 🟩 Correct letter, correct position  
  - 🟨 Correct letter, wrong position  
  - ⬜ Incorrect letter  
- Keyboard Interaction: Type directly using your keyboard  
- Dynamic Word List: Words are randomly selected each game  
- Clean UI: Minimal and responsive design  

---

## 🎮 How to Play

1. Guess the hidden word (usually 5 letters)  
2. Press **Enter** to submit your guess  
3. Use the color hints to refine your next guess  
4. Find the word before you run out of attempts  

---

- Frontend: HTML5, CSS3, JavaScript  
- Logic: Vanilla JavaScript
